* Ruby version: 3.2.0
* Rails version: 7.0.4

* Get start:

  - Setup:
    + bundle install (always run after pull code)

  - Database:
    + Change mysql password at: config/database.yml
    + Create database run: rails db:create db:migrate

  - Start server:
    + To start server (at localhost:3000) run: rails s
